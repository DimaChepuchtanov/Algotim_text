# Тут полное решение 1 практической по Алгоритмам и структурам данных. 

Задачи фибаначи расписан в 2х экземплярах: 
  1) Тот, что не закоментирован - это простой алгоритм, не оптимизированный, выполняется медленнее, чем его аналог
  2) Второй это тот, что спрятан в комменте, он быстрее выполняет свою задачу. 
   
Определитель матрицы работает по схеме: 
 Если матрица 1 - то первое число это и есть определитель 
 Если матрица 2х2 то определитьель считается умножением 11 на 22 и вычитаением 12*21
 Если матрица больше 2х2, то просчет происходит по методу Гауса ( для матриц больше 5х5 есть погрешнать в вычислении на 5-6 от реального ответа)
